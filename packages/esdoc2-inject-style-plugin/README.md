# esdoc2 Inject Style Plugin
## Install
```bash
npm install esdoc2-inject-style-plugin
```

## Config
```json
{
  "source": "./src",
  "destination": "./doc",
  "plugins": [
    {"name": "esdoc2-inject-style-plugin", "option": {"enable": true, "styles": ["./foo.css"]}}
  ]
}
```

`enable` is default `true`.
