# esdoc2 Unexported Identifier Plugin
## Install
```bash
npm install esdoc2-unexported-identifier-plugin
```

## Config
```json
{
  "source": "./src",
  "destination": "./doc",
  "plugins": [
    {"name": "esdoc2-unexported-identifier-plugin", "option": {"enable": false}}
  ]
}
```

`enable` is default `false`.
