**We are waiting for your pull request!**

## Install
```bash
npm install esdoc2-publish-markdown-plugin
```

## Config
```json
{
  "source": "./src",
  "destination": "./doc",
  "plugins": [
    {"name": "esdoc2-publish-markdown-plugin"}
  ]
}
```
