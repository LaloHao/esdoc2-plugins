# esdoc2 External Node.js Plugin
## Install
```bash
npm install esdoc2-external-nodejs-plugin
```

## Config
```json
{
  "source": "./src",
  "destination": "./doc",
  "plugins": [
    {"name": "esdoc2-external-nodejs-plugin", "option": {"enable": true}}
  ]
}
```

`enable` is default `true`.
